# Jakarta JSON Binding TCK Distibution

This bundle contains the Jakarta JSON Binding TCK. The contents of this bundle are:
- artifacts/* - the TCK tests jar and sample script to install into a local maven repo
- docs/*      - the TCK user guide (HTML and PDF)
- bin/*       - a TCK starter project with sample code from user guide
- README.md   - this README
- LICENSE     - the EFTL license terms

## Release Notes

### Facts

The test suite bundle contains the following:
- 295 total tests

### Platform

The Jakarta JSON Binding TCK tests have been built and tested running on the Java(TM) Standard Edition 17 (Java SE 17) platform.

The Jakarta JSON Binding TCK tests run on the following platforms:
- ubuntu-latest

### Installing and Running

Please refer to the Jakarta JSON Binding TCK User's Guide in the docs/ directory for complete instructions on installing, setting up, and running the Jakarta JSON Binding TCK.
